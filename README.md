# Gunpla Backlog Manger

A React + TypeScript app to manage your Gunpla backlog.

## Features

- Add, edit, and delete Gunpla builds
- Organize your building priorities
- Monitor your progress
- Export your backlog as JSON or CSV

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
npm install
```
